getUserProfileInsightCombinations(profileId, options, callbackopt) → {IntellogoResponse.<UserProfilesAPI~ProfileCombination>}

Retrieve combinations of insights matching the user profile. These combinations can be used to create smart collections relevant to the user profile.
Each combination constists of two or more insights that matched enough of the profile's content. Results are sorted by the number of profile content matched (descending). Typically there are many relevant insight combinations, so it is advised to use the pagination parameters.
Note: This method will return results only if the user profile has autoupdateCombinations set to true and a profile update task has finished since the last profile update.

CombinationOptions

Type:
  • Object
Properties:
Name Type Description
combinationCategoriesMin Number

The minimum number of insights in a combination (Only combinations containing at least this number of insights will be returned.)
combinationCategoriesMax Number

The maximum number of insights in a combination. (Only combinations containing at most this number of insights will be returned.)
combinationContentsMinCount Number

The minimum number of content that a combination must match. Applied only when combinationContentsMinPercent is NOT specified.
combinationContentsMinPercent Number

Acts like combinationContentsMinCount, but instead of setting a number of content, specifies what percentage of the profile's content must be matched by each returned combination
tags Array.<String>

Filter returned combinations by tags.
A combination matches a tag if at least one of the combination's insights has that tag.
Only combinations that match all specified tags will be returned
from Number

Start of range. Used for pagination
to Number

End of range.Used for pagination
