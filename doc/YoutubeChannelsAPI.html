importChannel(channelUrl, importOptions, callbackopt)

Import all videos from a specified Youtube channel. Accepts the same arguments as YoutubeChannelsAPI#initiateChannelImport, except the last argument must be a callback function. The callback will be called once the task has finished. Task status will be checked by polling.

ImportParams

Channel Import Params

Type:
  • Object
Properties:
Name Type Attributes Description
autoSub Boolean <optional>

Whether to use the automatically generated Youtube subtitles, or only user-uploaded subtitles
refreshAll Boolean <optional>

If true, Intellogo will try to import all videos. Otherwise, only recent videos (from last week) will be imported.
enableCaptionsChunking Boolean <optional>

If true, enables the experimental chunking behavior. The captions will be split in chunks of the specified length and they will be separately imported. The video captions as a whole will still be imported.
chunkSize Number <optional>

If chunking is enabled, the chunk size to use
