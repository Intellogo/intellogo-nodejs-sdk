Class: IntellogoClient

IntellogoClient

new IntellogoClient(options)

Creates an instance of the IntellogoClient. All methods accept an optional callback argument as their last parameter.
All methods also return an EventEmitter that emits data events with the requested data, or an error event in case of an error. However, it is recommended to use the callback option.
Calling any of the methods will automatically get an OAuth2.0 access_token, which will be used for all subsequent requests. The access_token will also be automatically renewed in case it expires.
