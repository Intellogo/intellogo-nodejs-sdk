NotificationsWebService

A simple web service that can register callbacks for specific webhooks and triggers them when a POST request is received.

registerWebhook(fn) → {String}

Creates a webhook and registers the given function for it. When a request is sent to this service on the generated URL, the function will be called as a callback with two arguments: an error (falsy value if the task was executed successfully), and a result object.

Type Definitions

NotificationOptions

Options to use for setting up a local web service. The web service will be used only to improve the mechanism for notifications on finished tasks. Otherwise, a simple polling is used.
N.B.: Make sure that the specified ip:port combination is accessible from outside your local network, or task notifications may not work as expected
