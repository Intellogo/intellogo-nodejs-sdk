(private) _wrapTaskExecution(taskInitiator, executionParams)

Creates a wrapper around one of the task-initiating methods. When the wrapper is called, it initiates a task using the specified method and the provided arguments. The last argument is expected to be a callback, otherwise an error is thrown. The wrapper then waits for it to finish either by polling, or using a webook if the IntellogoClient is set up for it. The callback is called when the task has finished or if an error occurs.
Parameters:
Name Type Description
taskInitiator IntellogoFunction A method that can initiate a task. It should already be bound to the proper context.
executionParams TrainingAPI~TaskWrapperParams Parameters controlling how the wrapper is set up.

generateContentRecommendations()

Generate recommendations for a content. This method initiates a content task and waits until the task has completed to call its callback. It accepts the same arguments as TrainingAPI#initiateContentRecommendationGeneration except that the last argument must be a callback function. The callback will be called when the task has finished. If a local notifications web service has been initialized (using IntellogoClient#initNotificationsService, a webhook will be used to signal task completion. Otherwise, regular polling will be used.
