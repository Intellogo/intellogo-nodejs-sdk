ContentFilter

Describes a content filter. The commonly used filters are documented below. You can add custom key-value pairs to this object. In that case, the system will try to find content that has exactly matching metadata. Such queries might be slower and/or return no results, so use this feature with caution.
Do not specify sources and sourceGroups at the same time - one of them will be overwritten!
Type:
  • Object
Properties:
Name Type Attributes Description
source Array.<String> <optional>
 only match content from these sources
sourceGroup Array.<String> <optional>
 only match content from these source groups. Supported source groups are 'wikipedia', 'web', and 'books'
acquisitionDate Number <optional>
 A timestamp (measured in seconds). If specified, only content newer than this will be matched
